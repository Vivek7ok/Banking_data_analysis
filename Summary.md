#### Project summary

###### 

###### Objective

The main goal of this project was to find out why the bakery business was making a loss. After finding the reasons, I gave some simple recommendations to help reduce the loss and increase the profit.



###### Steps

Collected the data.

Cleaned the data using Excel.

Used SQL to find important insights and the main reasons for the loss.

Built a Power BI report to show the results in a simple and easy-to-understand way.

Shared recommendations that can help improve the business and increase profit.





You can find all the project files in this GitHub repository. The full analysis, key findings, and recommendations are available in the README file.
